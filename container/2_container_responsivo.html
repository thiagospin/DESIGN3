O container-sm aplica largura fixa a partir de 576px, enquanto abaixo disso o container é 100% fluido.
O container-md define largura fixa somente a partir de 768px, permanecendo fluido em dispositivos menores.
O container-lg ajusta a largura fixa a partir de 992px, ideal para notebooks e desktops menores.
O container-xl fixa a largura a partir de 1200px, aproveitando bem o espaço em monitores grandes.
O container-xxl mantém a largura fixa a partir de 1400px, sendo adequado para telas muito amplas e resoluções elevadas.
