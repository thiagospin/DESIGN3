Classe "conatainer" simples vai mudando conforme o atingimento dos breakpoints.

No Bootstrap 5.3, o breakpoint sm (small) é ativado a partir de 576px de largura de tela, permitindo que o container tenha largura fixa acima desse valor.

O breakpoint md (medium) é aplicado a partir de 768px, ajustando o layout para telas de tamanho médio, como tablets em orientação paisagem.

O breakpoint lg (large) entra em ação a partir de 992px, otimizando a apresentação para notebooks e desktops de menor resolução.

O breakpoint xl (extra large) começa a partir de 1200px, indicado para monitores maiores, mantendo um espaçamento equilibrado no conteúdo.

O breakpoint xxl (extra extra large) é utilizado a partir de 1400px, garantindo que o container aproveite melhor o espaço em telas muito amplas.
